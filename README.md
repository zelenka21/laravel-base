# Laravel Boilerplate

## Stack
- Laravel 13
- MySQL (Docker)
- Redis (Docker)
- Nginx (Docker)


## Setup

* Clone the repository using `git clone git@github.com:zelenka21/laravel-base.git` command.
* Start Docker environment using `make start` command.
* Install dependencies using `make install` command.

## Available commands

* `make start` start the containers.
* `make stop` stop the containers.
* `make exec-php` enter the php container to execute commands, run composer etc.
